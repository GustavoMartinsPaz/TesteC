            --operadores--
Os operadores aqui em C são bem parecidos com os usados na aritmética normal:

    '+' -> Adição
    '-' -> Subtração
    '*' -> Multiplicação
    '/' -> Divisão

    As contas feitas possuem precedência parecidas com a matemática que você está acostumada

    // Faça um código que soma 10 com 20
    // Faça outro que multiplique 2 por 3
    // Faça um que pergunte ao usuário 2 números e some eles

    Depois de fazer isso você vai se acostumar a usar os operadores e a estrutura de C em si (agora sei explicar essa partekkk)

Temos outros operadores:

    '++' -> Incremento, Exemplo: i++   isso seria i = i + 1
    '--' -> Decremento, Exemplo: i--   isso seira i = i - 1

Operadores de comparação, esses serão muito usados em loops de do, while e for, e em if and else:

    '==' -> Igual a
    '!=' -> Diferente de
    '>' -> Maior que
    '<' -> Menor que
    '>=' -> Maior ou Igual
    '<=' -> Menor ou igual

Também temos os operadores lógicos, esses ainda não peguei bem o jeito, mas com o tempo eu te explico:

    '&&' -> E lógico
    '||' -> OU lógico
    '!' -> NÃO lógico

Operadors de atribuição:

    '=' -> Atribuição simples         int a = 10;
    * Atribuição composta:
        '+='       a += 5  é      a = a + 5
        Mesma lógica para '-=', '*=' e '/='

Acho que isso é tudo sobre operadores
